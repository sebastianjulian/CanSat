Project Overview

TRITON represents the evolution of our satellite simulation project into a sophisticated autonomous submarine navigation system. Designed for real-world underwater operations, TRITON integrates environmental monitoring with inertial navigation to provide comprehensive submarine guidance and mission-critical data collection capabilities.

🌊 Environmental Monitoring

Advanced BME280 sensor provides precise temperature, humidity, pressure, and altitude measurements. Pressure readings enable accurate depth calculation for underwater navigation, while environmental data supports mission planning and safety protocols.

🧭 Inertial Navigation

MPU6050 inertial measurement unit delivers 6-axis motion sensing with 3-axis accelerometer and 3-axis gyroscope. This provides real-time orientation tracking, motion detection, and navigation support for autonomous submarine operations.

📡 Wireless Communication

LoRa long-range wireless technology enables reliable data transmission between submarine and surface operations. Real-time sensor data streaming supports remote monitoring and mission control from considerable distances.

📊 Real-time Dashboard

Web-based mission control interface provides live visualization of all sensor data, statistical analysis, and comprehensive logging. Chart.js integration delivers professional-grade data visualization for operational analysis.

💾 Data Logging

Comprehensive CSV logging with automatic archiving ensures mission data integrity. Threshold-based data transmission reduces network overhead while maintaining critical data resolution for post-mission analysis.

⚡ Autonomous Operation

Raspberry Pi-based sensor collection with intelligent error handling and graceful degradation. System continues operation even with partial sensor failures, ensuring mission continuity and data collection reliability.

Technical Specifications

Hardware Platform

Controller: Raspberry Pi Zero 2 W
Environment: BME280 (I2C 0x76)
IMU: MPU6050 (I2C 0x68)
Communication: LoRa 9600 baud

Sensor Capabilities

Temperature: ±0.25°C resolution
Pressure: ±0.5 hPa threshold
Acceleration: ±16g range
Gyroscope: ±2000°/s range

Data Management

Logging: CSV with MET timestamps
Update Rate: ~1 second intervals
History: 300 data points per metric
Web Interface: Port 5000, network accessible

Mission Capabilities

TRITON is engineered for comprehensive submarine operations with advanced monitoring and navigation capabilities. The system provides continuous environmental monitoring for depth calculation through pressure readings, inertial navigation support through accelerometer and gyroscope data, and real-time wireless data transmission for surface monitoring and control.

The dual-platform architecture ensures operational redundancy and reliability. The Raspberry Pi handles autonomous sensor collection and LoRa transmission, while the PC base station manages data reception, logging, statistical analysis, and web-based mission control interface. This design enables real-time monitoring of submarine operations from considerable distances while maintaining comprehensive data logging for post-mission analysis.
